# List all installed global packages

```sh
npm list -g --depth=0
```

# Remove all global packages

```sh
npm ls -gp --depth=0 | awk -F/ '/node_modules/ && !/\/npm$/ {print $NF}' | xargs npm -g rm
```

where:

- `npm ls -gp --depth=0`  
  List all installed packages
- `awk -F/ '/node_modules/ && !/\/npm$/ {print $NF}'`  
  print package dir
- `xargs npm -g rm`  
  remove package

## Required

- GitHub repository
- Netlify account
- Gatsby project

## Step

1. create new Nelify project
   Netlify will ask you to choose Git service
2. connect with GitHub
3. select your repository
4. config build
   it will automatically use `yarn` and branch `master`
5. done

when you push `master` into remote, Netlify will automatically build and deploy
